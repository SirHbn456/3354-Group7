Project Overview

The Calendar Software is a team project developed for the CS3354 Software Engineering course. The purpose of this application is to help users organize and manage their schedules through an easy-to-use digital calendar.

The software allows users to create, edit, and delete events while providing multiple calendar views and automatic time conflict detection. Additional features include recurring weekly events, event categories with color coding, and special highlighting for holidays and weekends.

 Planned Features

Monthly, Weekly, and Daily calendar views
Add, edit, and delete events
Time conflict detection
Weekly recurring events
Event categories
Color-coded events
Holiday and weekend highlighting
